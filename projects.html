Path Finder

Team project for CS 230 Data Structures (Fall 2014)

A grid-maze game in which a player must maneuver his/her piece to its target destination in the fewest number of steps possible (if multiple targets are present, the player must strategize in which order to visit each target to minimize steps traveled)

  • Written in Java (17 classes total)
  • Game incorporates graph algorithms (ie: breadth-first search), stacks, recursion, scanning in and printing out to text files, graphical user interface
  • Features include: undo and reset options (with keyboard shortcuts), completely randomized levels, customizable levels, computer hints
